# olist_ecommerce_analysis

基于巴西 **Olist** 电商数据集的完整数据分析项目，包含数据清洗、探索性分析、用户价值分层、销售趋势与可视化。

## 一、项目简介
本项目使用公开的巴西 **Olist** 电商平台真实交易数据，通过 Python 进行全流程数据分析，旨在挖掘用户消费行为、商品销售规律、区域分布特征及用户价值结构，为电商运营、营销策略和用户运营提供数据支撑。

## 二、技术栈

| 开发语言 | Python |

## 三、数据集说明
本项目使用巴西 **Olist** 电商平台公开数据集，包含以下核心业务维度：
- 订单信息：订单时间、订单状态、商品价格、运费等
- 用户信息：用户 ID、用户所在地区等
- 商品信息：商品品类、商品价格、商品重量等
- 支付信息：支付方式、支付分期、支付金额等
- 卖家与物流信息：卖家信息、配送时间、物流状态等

数据集总大小约 **58MB**，已通过 Git 进行版本管理。

## 四、核心分析内容
### 1. 数据概览与清洗
- 缺失值、异常值、重复值的识别与处理
- 数据类型转换、时间特征提取与格式标准化
- 多表关联与数据整合，构建完整分析表

### 2. 销售整体情况分析
- 订单量、销售额的时间趋势分析（日 / 月 / 年维度）
- 支付方式分布统计与占比分析
- 订单状态全流程统计与转化率分析

### 3. 用户行为分析
- 用户地域分布热力分析
- 用户复购率、留存率计算
- 用户消费频次、客单价分布与分层

### 4. 商品与品类分析
- 热销品类 TOP10 排名
- 商品价格区间分布统计
- 品类销量与利润贡献对比分析

### 5. RFM 用户价值分层
基于 **Recency**（最近一次消费时间）、**Frequency**（消费频次）、**Monetary**（消费金额）构建 RFM 模型
- 将用户划分为 6 大层级：重要价值用户、重要发展用户、重要保持用户、重要挽留用户、一般用户、低价值用户
- 输出各分层用户占比，并针对性给出运营策略建议

### 6. 物流与配送分析
- 全平台配送时长统计与分布
